# Day 3: Treasure Island

A text-based choose-your-own-adventure game where the player navigates a series of choices to find the hidden treasure - or meet an unfortunate end.

## What I learned
- Control flow with `if`, `elif` and `else`
- Nested conditionals
- Normalizing user input with `.lower()`
- Raw strings (`r'''...'''`) for preserving backslashed in ASCII art

## Example
```
Welcome to the Treasure Island.
Your mission is to find the treasure.
Do you want to go left or right? left
You come across river. Do you want to swim or wait? wait
There are three doors of different colors: red, blue, and yellow. Which door do you open? yellow
You found the treasure. Congratulations - you win!
```

## How to run
```bash
python main.py
```

# Day 4: Rock, Paper, Scissors

A command-line Rock, Paper, Scissors game where the player faces off against a randomized computer opponent. Choices and outcome are displayed using ASCII art.

## What I learned
- The `random` module (`random.randint`)
- Lists and indexing
- Boolean logic with multiple conditions (`and`, `or`, `not`)
- Using constants for readable code
- Early validation (guard clauses)

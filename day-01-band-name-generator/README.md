# Day 1: Band Name Generator

A simple program that generates a band name from two pieces of user input:
the city you grew up in and your pet's name.

## What I learned
- `print()` and `input()` functions
- Storing user inputs in variables
- String concatenation and f-strings
- Basic string methods (`.strip()`, `.title()`)

## How to run
```bash
python main.py
```

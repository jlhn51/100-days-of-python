# Day 2: Tip Calculator

A command-line tip calculator that splits a bill evenly among diners, including a customizable tip percentage.

## What I learned
- Data types (int and float)
- Type conversion
- Math operators and order of operations
- F-string formatting (`:.2f` for currency)

## Example
```
What is the total bill? $150
How much tip would you like to give? 10, 12, or 15? 12
How many people to split the bill? 5
Each person should pay: $33.60
```

## How to run
```bash
python main.py
```
